# FinSight — Smart Financial Insight Dashboard

FinSight adalah aplikasi web untuk membantu pelaku usaha kecil menghitung modal, keuntungan, dan estimasi waktu balik modal (Break-Even Point — BEP). Tersedia juga AI Assistant gratis (menggunakan model lokal / open‑source) untuk saran bisnis ringan, analisis sederhana, dan penjelasan laporan keuangan.

## Fitur Utama
- Input modal awal dan biaya operasional (harian / mingguan / bulanan)
- Input harga jual per produk dan jumlah penjualan per periode
- Perhitungan otomatis:
  - Total pemasukan
  - Total pengeluaran
  - Keuntungan bersih
  - Laba kotor
- Estimasi waktu balik modal (BEP) dan grafik progres balik modal
- AI Assistant untuk rekomendasi dan analisis sederhana

## Cara Menjalankan Aplikasi
1. Clone repository
   ```
   git clone https://github.com/usernamenuh/UMKMBoost-AI.git
   cd UMKMBoost-AI
   ```
2. Install dependensi frontend
   ```
   npm install
   npm run dev
   ```
3. Install dependensi PHP
   ```
   composer install
   ```
4. Jalankan server pengembangan (contoh)
   ```
   php artisan serve
   ```
   Atau jalankan sesuai setup proyek Anda (mis. docker, sail, dsb).
